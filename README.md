## Installation

Install my-project with npm

# Navigate to the Project
```bash
  cd react-dev-portfolio
```

# Install Dependencies
```bash
  npm install
```

# Start development server
```bash
  npm run dev
```
The portfolio will be available at http://localhost:5173/ by default.
